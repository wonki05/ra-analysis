## 1. 입력 파일

- 초안: `converted-solutions_GPTdraft` 해제본 (기존 `official-solutions/2014_solution_files`)
- PDF: `docs_solutions` 2014 LEET 추리논증 해설 (70p 렌더, 200dpi, `_page_renders/2014`)
- 검수보고서: `converted-solutions_GPTdraft/review-reports/2012-2014_RA_solutions_review_report.md` (2014 섹션)
- 페이지 오프셋: 렌더 pN = 책 (321+N)쪽. 출제방향 p002~003(책323~324), 문항해설 p004(책325, Q01)~p070(책391, Q35).
- **PDF는 스캔본(텍스트레이어 없음)** → 전 페이지 육안 대조.
