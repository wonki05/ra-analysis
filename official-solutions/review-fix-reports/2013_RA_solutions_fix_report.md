# 2013학년도 RA 공식해설 Markdown 수정 보고서

## 1. 입력 파일

- 초안: `converted-solutions_GPTdraft` 해제본 (기존 `official-solutions/2013_solution_files`)
- PDF: `docs_solutions` 2013 LEET 추리논증 해설 (82p 렌더, 200dpi, `_page_renders/2013`)
- 페이지 오프셋: 렌더 pN = 책 (392+N)쪽. 문항해설 렌더 p006(책398)~p082(책474).
- **PDF는 스캔본(텍스트레이어 없음)** → 전 페이지 육안 대조.

## 2. 처리 결과

- 출력 폴더: `official-solutions/2013_solution_files`
- Markdown 파일: `2013_RA_official+solutions.md`
- 이미지 파일 수: 2개 (08_1 신규 가계도, 13_1 기존)
- 문항 수: 35 (## 01 ~ ## 35, 연속)
- 최종 판정: **통과** (Q01~Q35 전면 재교정, 본문·표·수식·이미지 모두 PDF 원문 대조)
- 초안은 구조는 대체로 양호했으나 A/B/C 라벨·선지번호·ㄱㄴㄷ·기호·수식이 대다수 문항에서 OCR 손상 → 문항 단위 재작성 후 splice.

### 3-2. OCR 손상 복원 (대표)

- A/B/C·A~E 라벨: Q14(강좌배정 A~E), Q17(주장 A/B/C), Q23(네트워크 A~D), Q28(지금파/반지금파/손턴) 등 전 문항 대량 복원.
- 원기호·밑줄기호: Q11 ㉠~㉤(사이코패스 논증), Q10 ⓐ~ⓙ·G₁G₂G₃(지식정당화), Q21 ㉠㉡·(a)(b)(라케스 용기).
- 원칙/변수기호: Q18 원칙 p/q/r(이중효과), Q26 점 F/G/H/I·곡선 OP/QR·X₀(위험-기대이익 그래프).
- 판정기호: Q17 ㄱ(○)·ㄴ(×)·ㄷ(○) 복원.
- 화살표(→): 판단·평가형 문항(Q23~Q25, Q29~Q33 등)의 분석 표시 "→" 복원(초안은 ㅡ/ㆍ로 손상).
- 오탈자 교정(OCR): 전속→전족, 가날픈→가냘픈, 왜꼭→왜곡, 물리법직→물리법칙, 술한→숱한, 여러고→여럿이고, 될 수→뛸 수(Q32 물리), 이렇거→이렇게, 못하더러도→못하더라도, 있으드로→있으므로 등.

### 3-3. 구조 정정 (관례 일치: 제시문 해설=개요 / 선택지별·<보기> 해설=정답+오답)
