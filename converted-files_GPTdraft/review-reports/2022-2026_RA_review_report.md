# LEET 추리논증 문제지 Markdown 변환 검수결과 요약

## 검수 개요

- **작업대상 연도**: 2022년 ~ 2026년, 총 5개년
- **과목**: 추리논증, 약어 RA
- **원문 비교대상**: `[연도]_LEET_추리논증_문제지(홀수형).pdf`
- **검수대상**: `[연도]_RA_improved_outputs.zip`
- **주요 검수사항**
  - 문항 및 제시문의 글자·단어 누락 여부
  - 수식 누락 여부
  - 표 및 이미지 누락 여부
  - 볼드, 밑줄 등 강조표시 누락 여부
  - 문항번호 및 문항범위 기재 오류 여부
  - 기타 Markdown 변환 과정에서 문제될 수 있는 사항

## 사용자가 직접 확인하면 좋은 부분

- **공통**: 모든 이미지 문항에 들어간 `<이미지 포함됨>`은 원문에 없는 메타 문구입니다. GitHub 분석용 주석으로 의도한 것이 아니라면 삭제하는 것이 좋습니다.
- **2025년 28번**: `실험 2의 결과`의 **“의” 누락**은 실제 수정이 필요합니다.
- **2022~2024년**: 원문 PDF가 스캔 이미지라 자동 텍스트 추출이 불안정했습니다. 구조·문항 수·이미지 파일 존재 여부는 확인했지만, 글자 단위 완전성은 수식·한자·특수기호가 많은 문항을 중심으로 최종 육안 확인을 권합니다.
- **HTML table 사용 문항**: 2025년 2번·32번 등 일부 표는 HTML table로 들어가 있습니다. GitHub 렌더링 목적이면 대체로 가능하지만, 분석 파이프라인이 순수 Markdown 표만 처리한다면 별도 변환이 필요합니다.
