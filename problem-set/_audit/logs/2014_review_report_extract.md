출처: `2012-2016_RA_review_report.md`

## 2014년 RA

- **문항 수 및 문항 번호**
  - Markdown 파일에 **1번부터 35번까지 전 문항이 존재**합니다.
  - 문항 번호의 누락이나 범위 오류는 발견되지 않았습니다.

- **본문·선택지 텍스트**
  - 원문 PDF와 Markdown의 문항 본문·보기·선택지를 비교한 결과, 실질적인 글자·단어 누락은 발견되지 않았습니다.
  - 일부 문항은 원문의 표형 선택지 또는 행렬 구조가 Markdown 표로 재구성되어 있습니다.

- **이미지·도식**
  - 29번의 두 이미지가 압축파일에 포함되어 있고 Markdown에서 참조됩니다.
  - 원문 PDF의 도형 이미지와 대응됩니다.
  - 원문 PDF 내부에 작은 ㄱ자 형태의 이미지 조각이 일부 있으나, 이는 문제 도식이 아니라 PDF 렌더링/글리프성 요소로 보이며 별도 삽입 대상은 아닌 것으로 판단됩니다.

- **표·수식**
  - 35번 리그/경기 결과 표의 행·열과 수치가 원문과 일치합니다.
  - 26번 등 조건표가 포함된 문항도 내용 누락은 발견되지 않았습니다.

- **강조표시**
  - 11번, 22번의 밑줄 강조가 `<u>...</u>`로 보존되어 있습니다.
  - 31번의 절기명 볼드 강조도 Markdown에서 `**...**` 형태로 반영되어 있습니다.
