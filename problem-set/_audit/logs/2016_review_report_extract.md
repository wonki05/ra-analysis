출처: `2012-2016_RA_review_report.md`

## 2016년 RA

- **문항 수 및 문항 번호**
  - Markdown 파일에 **1번부터 35번까지 전 문항이 존재**합니다.
  - 문항 번호의 누락·중복은 발견되지 않았습니다.

- **본문·선택지 텍스트**
  - 본문, 보기, 선택지 텍스트는 원문 PDF와 대체로 일치합니다.
  - 일부 수식과 표는 Markdown/LaTeX 문법으로 변환되어 있어 원문과 시각적으로는 다르지만, 내용 누락은 발견되지 않았습니다.

- **이미지·도식**
  - 21번, 24번, 34번, 35번 이미지가 압축파일에 포함되어 있고 Markdown에서 정상 참조됩니다.
  - 원문 PDF의 해당 도식·그래프와 대응됩니다.
  - 이미지 누락은 발견되지 않았습니다.

- **표·수식**
  - 주요 수식과 표는 Markdown 내에 반영되어 있습니다.
  - 20번의 남성·여성 합격률 계산식 부분은 원문에 보이는 `**` 표지가 Markdown에도 반영되어 있습니다. 다만 GitHub 렌더러에서 의도치 않게 강조 기호로 해석되지 않도록 최종 렌더링 확인이 필요합니다.

- **강조표시**
  - 밑줄 처리된 기호·진술은 `<u>...</u>`로 보존되어 있습니다.
  - 의미 있는 강조표시 누락은 발견되지 않았습니다.
