## 2023년 RA

- **문항 구성**: `# 01`부터 `# 40`까지 40문항 모두 존재합니다.
- **이미지**: 19번 선택지 이미지 5개, 30번 그래프 이미지 1개, 32번 도식 이미지 1개, 34번 도식 이미지 1개가 모두 존재합니다.
- **이미지 대조**: 19번 문항의 5개 논증 구조 도식은 원문 PDF의 ①~⑤ 도식과 시각적으로 대응합니다.
- **누락 이미지/표**: 검수 범위에서 이미지 파일 누락은 확인되지 않았습니다.
- **문제되는 사항**:
  - 19번, 30번, 32번, 34번 문항에 원문에 없는 `<이미지 포함됨>` 문구가 각각 삽입되어 있습니다. 실제 원문에는 없는 메타 표식이므로 삭제 권장입니다.
  - 2023 원문 PDF도 텍스트 레이어가 없는 스캔형 PDF입니다. OCR 대조 결과가 충분히 안정적이지 않아, 수식·표·특수기호가 많은 문항은 최종 육안 확인이 필요합니다.
