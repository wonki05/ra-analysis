# 2015 RA 검수보고서 발췌

출처: `2012-2016_RA_review_report.md`

## 2015년 RA

- **문항 수 및 문항 번호**
  - Markdown 파일에 **1번부터 35번까지 전 문항이 존재**합니다.
  - 문항 번호 체계는 정상입니다.

- **본문·선택지 텍스트**
  - 전체 문항의 본문·선택지·보기 텍스트에서 명백한 누락은 발견되지 않았습니다.
  - 비교 과정에서 감지된 차이는 주로 Markdown 표 변환, 수식의 LaTeX 표기, 줄바꿈·공백 차이입니다.

- **이미지·도식**
  - 17번의 두 이미지와 26번의 이미지가 압축파일에 포함되어 있고 Markdown에서 정상 참조됩니다.
  - 원문 PDF의 해당 이미지와 대응됩니다.
  - 이미지 누락은 발견되지 않았습니다.

- **표·수식**
  - 5번 사례 배열 표, 31번의 `<표 1>`, `<표 2>`는 원문의 항목과 수치가 보존되어 있습니다.
  - 수식은 Markdown/LaTeX 형식으로 변환되어 있으나, 수식 자체의 누락은 발견되지 않았습니다.

- **강조표시**
  - 검수 범위 내에서 원문의 의미 있는 강조표시가 빠진 사례는 발견되지 않았습니다.
