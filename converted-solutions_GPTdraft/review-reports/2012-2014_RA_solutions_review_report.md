- 검수 대상: 2012-2014학년도 LEET 추리논증(RA) 공식 해설 Markdown 패키지

# 1. 2012학년도 RA

## 1.1 검토요약

2012년 파일은 문항 01-35의 섹션 자체는 모두 존재하고, 해설에 필요한 주요 이미지 4개도 Markdown에서 참조된다. 그러나 텍스트 충실도는 낮다. 특히 선택지별 해설의 번호·라벨, A/B/C/D류 기호, 집합·수식·도식, 한글 단어가 OCR 결과 그대로 남은 부분이 많아 GitHub 업로드용 원문 전사본으로 사용하기 어렵다.

출제방향의 하위 제목도 원문과 다르게 재구성되어 있다. 원문은 `1. 문항 유형`, `2. 난이도`, `3. 문항 출제 시의 유의점 및 강조점` 구조인데, Markdown은 `1. 출제의 기본방향`, `2. 출제 범위 및 문항 구성`, `3. 난이도`, `4. 출제 시 유의점`으로 나뉘어 있어 공식 해설지의 제목 체계를 그대로 옮긴 것으로 보기 어렵다.

### 원문 이미지 재판독 기반 개선안

#### 19번 직급 도식

현재 L1206-L1238 부근의 도식은 아래처럼 코드블록으로 보존하는 방식이 적절하다. 아래는 원문 이미지 재판독 기반의 구조 개선안이다.

```text
C
|
F-D-E

    A
    |
C   B
|   |
F-D-E

(1) G가 C와 같은 직급인 경우:
    A
    |
G-C-B
  | |
  F-D-E

(2) G가 C보다 직급이 한 등급 높은 경우:
G   A
|   |
C-B
| |
F-D-E
```

#### 26번 `t1`-`t5` 및 집합식

현재 L1614-L1636은 다음과 같이 복원하는 것이 적절하다.

```text
t1 : 구입
 t2 : 동일 부품 변화
 t3 : 소프트웨어 업그레이드
 t4 : 외형 변화
 t5 : 복제

갑: {t1, t2, t3, t4} ≠ {t5}
을: {t1} ≠ {t2, t3} ≠ {t4} ≠ {t5}
병: {t1, t2} ≠ {t3, t4, t5}
```

### 추론 기반 개선안

- 문항 전반에서 `@`, `(@`, `66`, `690`, `60092` 등은 대체로 선택지 번호와 `정답 해설/오답 해설` 라벨이 깨진 것이다. 원문을 보며 각 선택지 번호를 복원해야 한다.
- `^`, `8`, `0`, `6`, `ㅁ`, `[`, `<` 등은 A/B/C/D 또는 ㄱ/ㄴ/ㄷ류 기호가 오인식된 사례가 많다. 문맥상 추정은 가능하나, 문항마다 변수가 다르므로 일괄 치환하면 안 되고 원문 대조가 필요하다.

## 1.6 사용자가 직접 확인했으면 하는 부분

2012년 파일은 오류가 산발적이지 않고 구조적으로 반복된다. 일부 문항만 수정하는 방식보다 PDF 원문을 기준으로 전체 Markdown을 재전사하거나, 최소한 문항별 선택지 해설 라벨·기호·수식·표를 전수 교정한 뒤 업로드하는 것을 권장한다.

# 2. 2013학년도 RA

## 2.1 검토요약

2013년 파일도 01-35번 문항 섹션은 모두 존재한다. 그러나 2012년보다는 적지만, 공식 해설로 보기 어려운 OCR 오류가 여러 문항에 남아 있다. 가장 큰 결함은 **08번 해설의 가계도 이미지 누락**이다. 원문 PDF에는 조전남의 가계도가 도식으로 제시되지만, Markdown에는 `|` 한 줄과 일부 텍스트만 남아 있어 도식 정보가 사실상 빠져 있다.

## 3.1 검토요약

2014년 파일은 세 연도 중 가장 양호하다. 01-35번 섹션이 모두 존재하고, Markdown 이미지 2개도 실제 파일로 존재하며 crop 상태도 양호하다. 본문에서 2012·2013년에 보이는 대규모 OCR 잡음은 발견되지 않았다.
